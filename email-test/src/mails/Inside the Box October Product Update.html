Latest Product News

Dev Container support 📦 — In one of our biggest product releases of 2023, we introduced Dev Container spec in CodeSandbox. This is now the default spec for all CodeSandbox projects and you can use our brand-new DevTool to configure *any* environment in seconds 🔥

Plus, with the latest update to our VS Code extension, you can now connect to containers running inside the CodeSandbox microVMs using the Dev Containers extension. This brings improved support for repos and sandboxes configured with Dev Containers and gets rid of the VS Code "trust" modal.

CodeSandbox Enterprise 🏦 — We introduced CodeSandbox Enterprise as a new offering that brings scalability, availability, security and trust to our CDE—the only one in the market that spins up an environment in 1.5 seconds, is available 24/7, and is fully collaborative. Private beta now open!

Boxy AI gets GPT-4 🤖 — We have upgraded Boxy, your CodeSandbox AI assistant, to use GPT-4 by default. Based on our testing, this (as expected) greatly increases the quality of Boxy's responses. (Boxy is available to Pro subscribers.)

Full Changelog — This past month, we made a lot of improvements and fixes across the board but we also shipped some powerful new features. Check the changelog page for all the details on what we shipped 🚀
